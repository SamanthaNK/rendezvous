## Getting Started

### Sign Up

1. Click **"Sign Up"** on the homepage
2. Choose your role: **Discover Events** (attendee) or **Organize Events** (organizer)
3. Enter your name, email, and password
4. Select your interests (1-5 categories)
5. Choose your city and neighborhood
6. Verify your email
7. Done!

### Login

- Go to homepage and click **"Login"**
- Enter your email and password
- You're in!

## For Event Attendees

### Browse & Search Events

- **Homepage**: See featured and trending events
- **Browse Events**: Click "Browse Events" in navbar
- **Search**: Use the search bar (top right) to find events by name
- **Filter**: Use filters to narrow by category, date, price, or location

### Save Events

- Click the **heart icon** on any event card
- View all saved events in **"Saved Events"** in your menu
- Click again to unsave

### Attend Events

- Click **"Interested"** on an event to mark attendance
- See how many others are interested
- Get notifications about the event

### Leave Reviews

- After an event ends, click **"Leave a Review"**
- Rate (1-5 stars) and write your feedback
- Helps other users discover quality events!

### Follow Organizers

- Visit an event and click **"Follow"** on the organizer card
- Get updates on their future events

## For Event Organizers

### Create an Event

1. Click **"Create Event"** (top right menu)
2. Fill in event details:
   - **Title** - Event name
   - **Description** - What's it about? (support markdown)
   - **Date & Time** - When and how long
   - **Location** - City, neighborhood, address
   - **Category** - Event type
   - **Price** - Free or paid (set price)
   - **Capacity** - Number of attendees
   - **Images** - Upload 1+ photos
3. Add social media links (optional)
4. Save as **Draft** or **Publish**

### Manage Events

- Go to **"My Events"** in your menu
- See tabs: **Upcoming**, **Draft**, **Past**, **Cancelled**
- For draft events: Click **"Publish"** to go live
- **Edit**: Modify any live event
- **View**: See how it looks to users
- **Analytics**: Track views, interested attendees, reviews
- **Delete**: Remove the event

### Get Verified

- Go to **"Organizer Dashboard"**
- Click **"Get Verified"** (if eligible)
- Provide org info and documents
- Admin reviews and approves
- Show verification badge to users

## For Administrators

### Dashboard

- **"Admin Panel"** in navbar
- See key stats: users, events, engagement
- Charts showing event trends
- Pending moderation queue

### Manage Content

#### Flagged Events

- Review flagged events (AI-detected suspicious content)
- **Approve**: Remove flag if content is OK
- **Keep Flagged**: Maintain warning
- **Remove**: Delete event if it violates guidelines

#### User Reports

- See reports on events/users
- **Resolve**: Close report with notes
- Track resolution status

#### Verification Requests

- Review organizer verification applications
- **Approve**: Grant verified badge
- **Reject**: Provide feedback & improvement suggestions

#### All Sections

- Filter by status (pending, resolved, etc.)
- View details of flagged content
- Take moderation actions
- Keep platform safe & trustworthy

## Tips & Best Practices

### For Attendees

 **Do:**

- Add interests when signing up for better recommendations
- Leave honest reviews to help the community
- Save events you're interested in
- Follow organizers you love

 **Don't:**

- Report events unless there's a real problem
- Spam events with multiple "interested" marks

### For Organizers

 **Do:**

- Add clear, detailed descriptions
- Upload high-quality images
- Keep event info up-to-date
- Respond to attendee reviews
- Get verified to build trust

 **Don't:**

- Use misleading titles or descriptions
- Set capacity lower than actual
- Modify key details last minute
- Delete events without notice

## Troubleshooting

### Can't Login?

- Check your email/password
- Click "Forgot Password" to reset
- Try clearing browser cookies

### Event Not Showing?

- Event might still be in draft (organizer hasn't published)
- Check if it was removed for violating guidelines
- Try refreshing page

### Payment Issues?

- Payment features coming soon
- Currently supports free events only

### Report a Problem

- Use **"Report"** button on event pages
- Describe the issue clearly
- Admin will review within 24-48 hours
